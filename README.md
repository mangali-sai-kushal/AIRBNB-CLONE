# Airbnb Clone

A full-stack web application that replicates core Airbnb functionality, allowing users to browse, create, and manage property listings with a complete review and rating system.

## Features
- User authentication (signup/login)
- Create, read, update, delete property listings
- Leave and view reviews with star ratings
- Responsive UI with Bootstrap/CSS styling
- Input validation and error handling
- Flash messages for user feedback
- MongoDB database integration

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Frontend:** EJS templates, HTML5, CSS3, JavaScript
- **Authentication:** Sessions/Cookies

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file with MongoDB connection string
4. Run: `node app.js`
5. Visit `http://localhost` (your configured port)
